Chat Endpoint

Send a Tafseer Message

Ask a question about any Quranic verse and receive a contextual answer grounded in classical tafseer commentary. Conversations are stateful via thread_id.

POST https://chat-with-tafseer.vercel.app/chat
Send requests as application/json. The API returns contextual answers from the tafseer of the specified verse and resource.
Request Body
Parameter Type Description Required
resource_id integer ID of the tafseer resource. E.g. 169 for Ibn Kathir. required
verse_key string Chapter:verse notation. E.g. "1:1" for Al-Fatihah, ayah 1. required
message string The question or message to send about the verse. required
thread_id string Session identifier for multi-turn conversation continuity. optional
Response
200 OK
{
  "answer":           "According to the tafseer, Al-Fatihah means...",
  "resource_id":      169,
  "verse_key":        "1:1",
  "chapter_number":   1
}
Response Fields
FieldTypeDescription
answer string The tafseer-grounded answer to your question.
resource_id integer Echoed resource ID used for the response.
verse_key string Echoed verse key.
chapter_number integer Surah number derived from the verse key.
