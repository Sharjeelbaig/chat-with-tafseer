Chapter Metadata

Retrieve a structured list of all 114 Surahs.

GET /surahs returns a flat JSON array of all Quranic chapters with their Arabic names, English transliterations, and verse counts.

GET /surahs

Choose this endpoint when you need to build an index, populate a Surah selection menu, or reference chapter-level metadata.

When to Use This Endpoint

Good fit

  • Indexing or storing source tafseer by chapter.
  • Manual reader or editor workflows.
  • Bulk chapter inspection without model generation.

Wrong fit

  • Verse-level Q&A with follow-up replies.
  • Any feature that needs thread continuity.
  • A frontend flow where the user expects a natural-language answer.
Error Handling
Status When it happens
404 The tafseer resource is missing or the chapter number is invalid upstream.
502 Quran.com failed to respond successfully or the transport request failed.
